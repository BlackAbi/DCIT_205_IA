# Computer Science Department - Contact Us Webpage

## Overview

This repository contains the code for a simple Contact Us webpage designed for the Computer Science Department at [University Name]. The webpage provides essential contact information, office hours, location details, and an interactive map.

## How to Use

1. **Clone the Repository:**
   
   ```bash
   git clone https://github.com/yourusername/computer-science-contact-us.git
   ```

2. **View the Webpage:**

   Open `index.html` in your web browser to explore the Contact Us webpage.

## Page Sections

### Contact Information

- Displays the department's address and general email.

### Office Hours

- Shows the regular office hours from Monday to Friday.

### Location

- Provides information about the department's physical location.

### Map

- Displays an interactive map showcasing the department's position on campus.

## Styling

The webpage is styled using simple and clean CSS to achieve a professional and user-friendly appearance.
